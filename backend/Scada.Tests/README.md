# Scada.Tests (legado)

Este projeto esta aposentado como suite oficial de testes.

Ele foi mantido apenas como referencia historica de testes manuais de conexao. Parte dos arquivos ainda usa namespaces antigos de drivers e nao deve ser usada como validacao automatizada do produto.

Fluxo suportado atualmente:

- regras MES: `backend/Scada.MesTests`
- testes manuais de protocolo: projetos simples dedicados em `backend/*SimpleTest`

Nao adicione novas regras de negocio aqui.
